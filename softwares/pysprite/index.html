pysprite

A Python 3 Re-Implementation of the Sample Parameter Reconstruction via Iterative TEchniques (SPRITE) procedure.

This library implements the SPRITE procedure in Python 3. To find out how to use the library for error detection and data forensics, visit the github repository.

The SPRITE procedure, and the original code on which this library is based, were developed by James Heathers, Jordan Anaya, Tim van der Zee and Nicholas JL Brown.
